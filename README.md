# 项目简介

## 核心功能
- 提供基于YOLOv8的道路实例分割检测系统完整源码
- 包含2500张标注数据集，涵盖道路类别
- 提供训练好的模型及训练结果
- 支持图片、视频及摄像头进行检测
- 支持选择不同模型
- 界面实时显示目标位置、目标总数和置信度信息
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 提供自定义置信度和IOU阈值设置
- 可选择是否显示标签和原图

## 适用场景
- 道路交通分析与监测
- 智能交通系统开发
- 计算机视觉研究与教学
- 道路维护与管理
- 实时监控系统应用

## 技术特点
- 基于Python 3.9开发
- 使用PyCharm+Anaconda或VSCode+Anaconda作为开发环境
- 集成OpenCV、PyQt5、Ultralytics、PyTorch等深度学习框架
- 提供图形用户界面，操作便捷
- 支持多种输入源处理（图片、视频、摄像头）
- 包含完整的项目文件和运行教程
